# Smart Expense Tracker 

Smart Expense Tracker is a Python-based CLI application that helps users manage their daily expenses efficiently.

## Features
- Add daily expenses
- View all expenses
- Category-wise expense summary
- Data stored in CSV file

## Technologies Used
- Python
- CSV File Handling

## How to Run
1. Clone the repository
2. Run the file:
   python expenses_tracker.py

## Use Case
Perfect for beginners learning Python and real-world logic implementation.
